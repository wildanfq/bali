**BALI** adalah proyek sistem operasi eksperimental berbasis **Microkernel** yang dibangun dari nol (*from scratch*) menggunakan bahasa pemrograman **Zig (v0.16.0)**. Proyek ini dirancang khusus sebagai materi penelitian dan pendidikan bagi pengembang yang ingin memahami mekanisme internal sistem operasi di atas arsitektur **RISC-V (RV32/RV64)**.
